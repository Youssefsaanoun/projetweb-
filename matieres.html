Matières Étudiées et Projets Réalisés

Matières Principales

Programmation Web I & II

Apprentissage des technologies web fondamentales : HTML, CSS, JavaScript. Développement de sites web statiques et dynamiques, maîtrise des frameworks modernes.

Développement Backend

Étude approfondie des architectures backend avec Spring Boot, .NET Core, et Node.js. Création d'APIs RESTful et gestion des bases de données.

Bases de Données

Conception et gestion de bases de données relationnelles et NoSQL. Modélisation, requêtes SQL complexes, et optimisation des performances.

Algorithmes et Structures de Données

Étude des algorithmes classiques, complexité algorithmique, et structures de données avancées pour résoudre des problèmes informatiques complexes.

Développement Mobile

Introduction au développement d'applications mobiles, avec focus sur les technologies cross-platform et les frameworks modernes.

Méthodes Agiles

Apprentissage des méthodologies agiles, Scrum, Kanban, et gestion de projet itératif. Mise en pratique lors de projets d'équipe.

Projets Réalisés

Plateforme E-commerce Gaming avec Spring Boot (Java)

Développement d'une plateforme e-commerce complète dédiée à la vente de produits gaming. Le projet utilise une architecture sécurisée et scalable avec Spring Boot pour le backend et MySQL pour la base de données. Fonctionnalités incluant la gestion des utilisateurs, catalogue de produits, panier d'achat, et système de paiement.

Technologies: Spring Boot (Java), MySQL, HTML, CSS, JavaScript

Plateforme E-commerce avec .NET Core (C#)

Conception et développement d'une application e-commerce complète pour la vente de livres. Le projet met en œuvre une architecture moderne avec .NET Core, incluant des fonctionnalités avancées de recherche, filtrage, et recommandations personnalisées.

Technologies: .NET Core (C#), MySQL, Entity Framework

Plateforme E-commerce avec PHP

Développement d'une solution e-commerce dédiée à la vente de produits gaming utilisant PHP. Le projet inclut une interface utilisateur intuitive, gestion complète des commandes, et système d'authentification sécurisé.

Technologies: PHP, MySQL, HTML, CSS, JavaScript

Projets de Travaux Pratiques

Réalisation de multiples travaux pratiques couvrant différents aspects du génie logiciel : développement d'applications web, manipulation de bases de données, implémentation d'algorithmes, et développement d'interfaces utilisateur modernes et responsives.
